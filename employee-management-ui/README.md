# Employee Management System - Frontend

Frontend application for Employee Management System built using React, TypeScript, Redux Toolkit, RTK Query, and Tailwind CSS.

## Tech Stack

- React
- TypeScript
- Vite
- Redux Toolkit
- RTK Query
- React Router
- Tailwind CSS

## Features

### Authentication
- Login
- Protected routes
- Role-based UI access

### Dashboard
- Employee overview
- Department statistics
- Attendance summary

### Employee Module
- Employee listing
- Create employee
- Update employee
- Delete employee
- Employee details

### Department Module
- Department listing
- Manager assignment
- Department status

### Leave Module
- Apply leave
- View leave history
- Approve/reject leave
