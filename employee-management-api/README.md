# Employee Management System - Backend

Backend API for Employee Management System built using Node.js, Express, TypeScript, and MongoDB.

## Tech Stack

- Node.js
- Express.js
- TypeScript
- MongoDB
- Mongoose
- JWT Authentication
- Zod Validation
- Redis (optional)
- REST API

## Features

### Authentication
- User registration
- User login
- JWT access token
- Refresh token authentication
- Role-based authorization

### Employee Management
- Create employee
- Get employees
- Update employee
- Delete employee
- Employee profile management

### Department Management
- Create department
- Assign manager
- Update department
- Department listing

### Attendance Management
- Employee check-in
- Employee check-out
- Attendance history
- Working hours calculation

### Leave Management
- Apply leave
- Approve/reject leave
- Leave status tracking
