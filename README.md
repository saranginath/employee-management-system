# Employee Management System

Full-stack employee management application.

## Tech Stack

### Backend
- Node.js
- Express.js
- TypeScript
- MongoDB
- Mongoose
- JWT Authentication
- Zod Validation

### Frontend
- React
- TypeScript
- Redux Toolkit
- RTK Query
- Tailwind CSS

## Features

- User authentication
- Role-based access control
- Employee management
- Department management
- Attendance tracking
- Leave management
